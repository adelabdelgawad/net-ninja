# Code of Conduct

This project adopts the **Contributor Covenant v2.1** as its code of conduct.

The full text is available at:
https://www.contributor-covenant.org/version/2/1/code_of_conduct/

## Summary

We are committed to providing a welcoming and harassment-free experience for everyone. We expect all participants to be respectful, constructive, and inclusive.

## Enforcement

Instances of unacceptable behavior may be reported to the project maintainers at **conduct@netninja.dev**. All reports will be reviewed and investigated promptly and fairly.

## Attribution

This Code of Conduct is adapted from the [Contributor Covenant](https://www.contributor-covenant.org), version 2.1.
